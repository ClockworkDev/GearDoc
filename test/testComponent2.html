Rendering libraries

This package registers the following rendering libraries:

someProxyRenderingLibrary

This is a proxy rendering library that does something with the events and the fowards them to the next library in the pipeline.

Rendering pipeline

This package sets the rendering pipeline to:

  • someProxyRenderingLibrary
  • <previousRenderingPipeline>

where <previousRenderingPipeline> is the pipeline before this package was loaded.
