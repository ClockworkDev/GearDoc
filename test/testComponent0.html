Components

This package registers the following components:

foodTracker

Description

This component keeps track of the food you eat

Listeners

This component listens to the following events:

iAteAnIcecream

You can call this event to notify that you ate an icecream

iAteAHotdog

You can call this event to notify that you ate a hotdog

The event expects the following data:

{"ketchup":"<Did you pour some deliciuos sauce?>"}

Triggers

This component can trigger the following events:

foodGoalReached

This event will be triggered once you have eaten too much.

The trigger sends the following data:

{"values":[{"x":"<X value on the first thumsbtick>","y":"<Y value on the first thumsbtick>"},{"x":"<X value on the second thumsbtick>","y":"<Y value on the second thumsbtick>"}],"calories":"<The number of calories that you have swallowed>"}
